Pasos para ejecutar el juego.

1. Instalar las librerías necesarias para ejecutar.

pip install pygame
pip install opencv-python

2. Inicia la ejecución del juego.

En la consola de sistemas operativos Linux o en la terminal de Windows:
python batalla_espacial.py

En un entorno de desarrollo:
Dar click en el botón de ejecución

3. Empieza a jugar. Se requieren dos objetos para manipular las naves espaciales, uno de color amarillo y otro de color rojo. Ambos objetos deben tener un área mínima de 5 centímetros cuadrados.
